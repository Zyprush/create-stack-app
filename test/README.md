# Test Project

This directory contains an example project generated by `create-stack-app`.

## Purpose

This test project demonstrates the output of the CLI and can be used for:
- Manual testing of the CLI
- Verifying generated project structure
- Testing integrations

## Regenerating

To regenerate this test project:

```bash
cd ..
npm run build
node dist/cli.js test --yes --in-place
```

Or from the root:
```bash
npm run build
npm start -- test --yes --in-place
```

## Notes

- This is an example React + Vite project
- The `node_modules` directory is excluded from git (see .gitignore)
- This project is for testing/demonstration purposes only
